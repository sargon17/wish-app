# Wish App (codename)

This project aims to connect users with developers by creating a feature-request layer between them, so users can express their needs and developers can build what people actually want.

## Tech Stack

### Backend

- Convex

### Auth

- Clerk

### Frontend

- TanStack Start
- React
- Vite
- Astro
- Tailwind
- shadcn/ui

### Tooling

- Vite+
